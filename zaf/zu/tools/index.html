# Cost Averaging Calculator

Cost Averaging (Dollar Cost Averaging) Bitcoin is an accumulation strategy that involves buying a set value of Bitcoin Bitcoin at regular intervals over a long term period.

The Exonumia project is working on a cost averaging calculatoropen in new window which uses historically available data on Bitcoins price vs currency pairs across the African continent.

# Bitcoin Calendar

Bitcoin has has a few days worth remembering and celebrating in it's short history. The Exonumia project is looking to curate a Bitcoin calendaropen in new window people can easily access to be aware of these historic days.
